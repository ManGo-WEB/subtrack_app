# Миграции Supabase

## Установка схемы базы данных

1. Откройте ваш проект в Supabase Dashboard: https://app.supabase.com
2. Перейдите в раздел SQL Editor
3. Скопируйте содержимое файла `001_initial_schema.sql` и выполните его
4. Скопируйте содержимое файла `002_seed_services_catalog.sql` и выполните его

## Структура миграций

- `001_initial_schema.sql` - Создание всех таблиц, RLS политик и триггеров
- `002_seed_services_catalog.sql` - Заполнение каталога популярных сервисов

## Важные моменты

- После выполнения миграций обязательно проверьте, что RLS политики активны
- Убедитесь, что триггер `handle_new_user` создан и работает для автоматического создания профилей
- В production окружении используйте Supabase CLI для управления миграциями
